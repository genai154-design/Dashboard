## 분석 지침

응답은 지정된 JSON 스키마에 맞춰 작성하세요.

- **sentiment:** `"긍정"`, `"중립"`, `"부정"` 중 하나 (K-방산·국방 산업 관점)
- **sentimentScore:** 0~100 정수 (높을수록 긍정적 전망)
- **summary:** 핵심 동향을 2~3문장으로 요약
- **highlights:** 3~4개 항목
  - `opportunity` — 기회 (수출·성장·기술 우위 등)
  - `risk` — 리스크 (지정학·예산·규제·원가 등)
  - `watch` — 주목 (추적이 필요한 이슈)

분석 시 국외·국내 뉴스를 함께 고려하고, K-방산 수출과 국내 방위사업 동향을 연결해 인사이트를 도출하세요.
